- Real-time market analysis
- Multiple trading strategies
- Risk management

## Usage
This app requires API keys to be set in the Space settings:
- ANTHROPIC_API_KEY
- POLYGON_API_KEY
- FRED_API_KEY
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID
